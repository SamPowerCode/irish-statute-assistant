# uv Migration Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace pip + `requirements.txt` with uv, moving all deps into `pyproject.toml` and generating a committed `uv.lock`.

**Architecture:** All 14 dependencies move verbatim from `requirements.txt` into `[project] dependencies` in `pyproject.toml`. The setuptools build backend is unchanged. `uv sync` generates `uv.lock` which is committed. `requirements.txt` is deleted. README updated to use `uv sync`.

**Tech Stack:** uv, pyproject.toml (setuptools backend)

## Chunk 1: Migrate deps and verify

### Task 1: Update pyproject.toml, .gitignore, and delete requirements.txt

**Files:**
- Modify: `pyproject.toml`
- Modify: `.gitignore`
- Delete: `requirements.txt`

There are no application tests to write for this task — it is pure config. Verification is done by running `uv sync` and `uv run pytest` in Task 2.

- [ ] **Step 1: Add dependencies to pyproject.toml**

Open `pyproject.toml`. The current `[project]` section looks like:

```toml
[project]
name = "irish-statute-assistant"
version = "0.1.0"
requires-python = ">=3.10"
```

Add the `dependencies` field so it becomes:

```toml
[project]
name = "irish-statute-assistant"
version = "0.1.0"
requires-python = ">=3.10"
dependencies = [
    "langchain>=0.3",
    "langchain-anthropic>=0.3",
    "langchain-core>=0.3",
    "pydantic>=2.0",
    "pydantic-settings>=2.0",
    "httpx>=0.27",
    "beautifulsoup4>=4.12",
    "tenacity>=8.0",
    "pytest>=8.0",
    "pytest-httpx>=0.30",
    "python-dotenv>=1.0",
    "langchain-chroma>=0.1",
    "langchain-huggingface>=0.1",
    "sentence-transformers>=3.0",
]
```

Leave everything else in the file unchanged.

- [ ] **Step 2: Add .venv/ to .gitignore**

Open `.gitignore` and add `.venv/` at the end:

```
.worktrees/
__pycache__/
*.pyc
*.pyo
.pytest_cache/
*.egg-info/
data/chroma/
.venv/
```

- [ ] **Step 3: Delete requirements.txt**

```bash
rm requirements.txt
```

- [ ] **Step 4: Commit**

```bash
git add pyproject.toml .gitignore
git rm requirements.txt
git commit -m "chore: migrate deps from requirements.txt to pyproject.toml for uv"
```
